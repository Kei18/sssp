## Setup

```sh
git clone https://github.com/Kei18/mrmp.git
cd mrmp
julia --project=.
(MRMP) pkg> instantiate
```

## Usage

#### Open JupyterLab

```sh
julia --project=. -e "using IJulia; jupyterlab()"
```
